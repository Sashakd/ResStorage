Вариант 3

    Заполнить массив простыми числами. 

    Реализовать функцию 

int insert(char *str, char* sub_str, int start, char *str_res, int res_len);

которая помещает в строку str_res исходную строку str, в которую вставлена подстрока sub_str, начиная с позиции start.

Буфер под результирующую строку (str_res) выделяется вызывающей стороной. Предусмотреть возможность возврата функцией insert размера результирующей строки str_res (включая признак конца строки).
